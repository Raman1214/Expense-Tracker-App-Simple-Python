# 💰 Expense Tracker App (Python)

A simple **Command-Line Expense Tracker** built using **Python**. This project allows users to record daily expenses, view all saved expenses, and calculate the total amount spent. It is a beginner-friendly project that demonstrates the use of Python lists, dictionaries, loops, conditional statements, and user input.

---

## 📌 Features

* ➕ Add new expenses
* 📅 Store expense date
* 🏷️ Categorize expenses (Food, Travel, Makeup, Books, etc.)
* 📝 Add expense descriptions
* 💵 Store expense amount
* 📋 View all recorded expenses
* 🧮 Calculate total expenses
* 🚪 Exit the application safely

## 📖 How It Works

The application stores each expense as a **dictionary** containing:

* Date
* Category
* Description
* Amount

All expense dictionaries are stored inside a **list**, making it easy to manage and calculate total expenses.

Example:

```python
{
    "date": "26-06-2026",
    "category": "Food",
    "description": "Lunch",
    "amount": 250
}
```
